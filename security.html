SECURITY
OVERVIEW

Bank-Level Security for South African Law Firms

  • 1. Data Residency & Sovereignty
  • All data is hosted exclusively in South Africa on AWS Johannesburg (af-south-1) region.
    No data is transferred outside SA borders, ensuring full POPIA compliance and protection from foreign jurisdiction risks.
  • 2. Encryption Standards
  • • In-transit: TLS 1.3 with Perfect Forward Secrecy
    • At-rest: AES-256 encryption with customer-managed keys
    • Database fields: Additional field-level encryption for sensitive data
    • Automatic key rotation every 90 days
  • 3. Access Controls & Authentication
  • • Mandatory Multi-Factor Authentication (MFA) option for all users
    • Role-Based Access Control (RBAC) with granular permissions
    • Zero-trust architecture: Every request verified
    • Session timeouts and automatic logout after 15 minutes inactivity
    • IP allow-listing available on Enterprise plans
  • 4. Monitoring & Threat Detection
  • • 24/7 real-time monitoring with SIEM (Security Information & Event Management)
    • Automated intrusion detection and prevention systems
    • Continuous vulnerability scanning (daily)
    • Regular independent security audits and penetration testing
    • Full audit logs retained for 12 months
  • 5. Incident Response & Breach Protocol
  • In the unlikely event of a security incident:
    • Immediate containment and forensic investigation
    • Notification to affected Firms and Information Regulator within 72 hours (POPIA requirement)
    • Transparent post-incident report and remediation plan
    • Dedicated incident response team on standby
  • 6. Compliance & Certifications
  • • 100% POPIA compliant with registered Information Officer
    • Aligned with ISO 27001 Information Security Management standards
    • PCI DSS compliant payment processing via PayFast
    • Annual third-party security audits and SOC 2 Type I readiness
    • Secure backups with 30-day retention and off-site redundancy (still in SA)
  • 7. Physical & Infrastructure Security
  • Underlying AWS Johannesburg data centres feature:
    • Biometric access controls
    • 24/7 on-site security
    • Redundant power and cooling
    • Fire suppression and environmental monitoring
  • 8. Your Responsibilities
  • To maintain maximum security, Firms should:
    • Use strong, unique passwords
    • Enable MFA on all accounts
    • Train staff on phishing awareness
    • Report suspicious activity immediately to security@lexpilot.co.za
