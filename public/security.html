SECURITY
OVERVIEW

Bank-Level Security for South African Law Firms

  • 1. Data Residency & Hosting
  • All data is hosted exclusively in South Africa (Johannesburg region) on secure cloud infrastructure. No data leaves the country, ensuring full POPIA compliance and zero cross-border risks.
  • 2. Encryption Standards
  • • End-to-end encryption for all data in transit (TLS 1.3).
    • Encryption at rest (AES-256) for stored data.
    • Secure key management with automatic rotation.
  • 3. Access Controls & Authentication
  • • Multi-factor authentication (MFA) available for all accounts.
    • Role-based access control (RBAC) — only authorised users access features based on tier.
    • Least-privilege principle: Team members have limited, audited access.
    • Session management: Automatic logout on inactivity.
  • 4. Security Monitoring & Audits
  • • Real-time threat detection and intrusion prevention.
    • Annual independent security audits by certified professionals.
    • Continuous vulnerability scanning and patching.
    • 24/7 monitoring for anomalies.
  • 5. Breach Response
  • In the unlikely event of a breach, we notify affected Firms and the Information Regulator within 72 hours (as required by POPIA). We provide full assistance for mitigation and recovery.
  • 6. Compliance Certifications
  • • Full POPIA compliance.
    • Alignment with ISO 27001 standards.
    • Regular penetration testing by ethical hackers.
    • Secure payment processing via PayFast (PCI DSS compliant).
  • 7. User Responsibilities
  • Firms must use strong passwords, enable MFA, and report suspicious activity immediately. Do not share credentials or input sensitive data without safeguards.
